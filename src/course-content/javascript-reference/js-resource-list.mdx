# JavaScript Resources

## References

Mozilla Developer Network (MDN) has comprehensive and reliable references for JavaScript in addition to tutorials.

<a href='https://developer.mozilla.org/en-US/docs/Web/JavaScript' target='_blank' class="mdxLink">
    Mozilla Developer Network
</a>

To research a specific topic, you can search for the topic plus mdn to find the specific MDN page (ie. "arrays mdn").
