## Activity I: Press the Button

Press the üçï button to add a pizza emoji to the emoji array.

## Activity II: Add a Button

Add a button to the app to add a different food item to the array.

```jsx title=App.js
// Before
<TouchableOpacity
  style={styles.button}
  onPress={() => updateFoodArray('üçï')}
>
  <Text style={styles.btnText}>üçï</Text>
</TouchableOpacity>

// After
<TouchableOpacity
  style={styles.button}
  onPress={() => updateFoodArray('üçï')}
>
  <Text style={styles.btnText}>üçï</Text>
</TouchableOpacity>
<TouchableOpacity
  style={styles.button}
  onPress={() => updateFoodArray('ü•ê')}
>
  <Text style={styles.btnText}>ü•ê</Text>
</TouchableOpacity>
```

## Activity III: Change the Parameter

Change the parameter sent to the updateFoodArray function.

```jsx title=App.js
// Before
<TouchableOpacity
  style={styles.button}
  onPress={() => updateFoodArray('üçï')}
>

// After
<TouchableOpacity
  style={styles.button}
  onPress={() => updateFoodArray('üçä')}
>
```

## Activity IV: Change the updateFoodArray

Delete the ...foodArray code from the updateFoodArray function.

What do you think this will change when you hit a button?

```jsx title=App.js
// Before
const updateFoodArray = (emoji) => {
  setFoodArray((foodArray) => [...foodArray, emoji]);
};

// After
const updateFoodArray = (emoji) => {
  setFoodArray((foodArray) => [emoji]);
};
```
