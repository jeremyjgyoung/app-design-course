## Code Explanation

Vibration is a component available from the Expo library.

```jsx
import { Video } from 'expo-av';
```

Here is the *Video* component from Expo. The source prop points to the location of the video to play.

```jsx
<Video
    source={{ uri: 'http://d23dyxeqlo5psv.cloudfront.net/big_buck_bunny.mp4' }}
    shouldPlay
    useNativeControls
    style={{ width: "100%", height: "50%" }}
/>
```
