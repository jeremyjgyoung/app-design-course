# Activity Set 1.3

This activity set will introduce you to the box model and more flexbox.

<a href="https://snack.expo.io/@jeremyjgyoung/activity-set-1.3" target="_blank" class="mdxLink">Activity Set 1.3 Code</a>

## Activity I: Change the Color

Change the *backgroundColor* of *boxOne* from "#0000ff" to *"#00ff00"*.

Which box is boxOne? How will the color change?

```jsx
// Before
backgroundColor: "#0000ff",
```

```jsx
// After
backgroundColor: "#00ff00",
```

## Activity II: Change the Text

Change the text within the first Text component.

```jsx
// Before
<Text style={styles.exampleText}>Here is the first example text.</Text>
```

```jsx
// After
<Text style={styles.exampleText}>Now I have changed the text.</Text>
```

## Activity III: Change the Padding

Next we will change the padding, border, and margin of the box. 
Before changing these properties, make a guess on how it will change the result.

Change the *padding* of *boxOne*.

```jsx
// Before
padding: "20px",
```

```jsx
// After
padding: "50px",
```

## Activity IV: Change the Border

Change the *borderWidth* and *borderColor* of *boxOne*.

```jsx
// Before
borderWidth: "5px",
borderColor: "#ff00ff",
```

```jsx
// After
borderWidth: "10px",
borderColor: "#ffff00",
```

## Activity V: Change the Margin

Change the *margin* outside of *boxOne*.

```jsx
// Before
margin: "50px",
```

```jsx
// After
margin: "2px",
```

## Activity VI: Change justifyContent

Change *justifyContent* for the parent *container* from *"center"* to *"flex-start"*.

```jsx
// Before
container: {
    flex: 1,
    backgroundColor: "#00ffff",
    alignItems: "center",
    justifyContent: "center",
},
```

```jsx
// After
container: {
    flex: 1,
    backgroundColor: "#00ffff",
    alignItems: "center",
    justifyContent: "flex-start",
},
```

## Activity VII: Add Padding, a Border, and Margin

Add *padding*, a *border*, and *margin* to *boxTwo*. Look at the boxOne as a reference.

```jsx
// Before
boxTwo: {
    backgroundColor: "#ff0000",
    width: "200px",
    height: "300px",
},
```

```jsx
// After
boxTwo: {
    backgroundColor: "#ff0000",
    width: "200px",
    height: "300px",
    padding: "5px",
    borderWidth: "1px",
    borderColor: "#000000",
    margin: "100px",
},
```
