## Code Explanation

The library used here is *'react-native-maps'*.

```jsx
import MapView from 'react-native-maps';
```

The *MapView.Marker* is a pin set at a specific coordinate that reveals a title and description when clicked on. You can even designate a hexcolor for your pin with the *pinColor* prop.

```jsx
<MapView.Marker
  coordinate={{latitude: 37.73538,
      longitude: -122.4324,}}
  title={"Title of the Marker"}
  description={"This is a description here."}
  pinColor='#ff00ff'
/>
```

Here is the code for showing a map starting with a particular region shown in the *initialRegion* prop.

```jsx
let App = () => {
  return (
    <View style={styles.mapContainer}>
      <MapView
        style={styles.mapStyle}
        initialRegion={{
          latitude: 37.78825,
          longitude: -122.4324,
          latitudeDelta: 0.0922,
          longitudeDelta: 0.0421,
        }}
      >
      </MapView>
    </View>
  );
};

let styles = StyleSheet.create({
  mapContainer: {
    flex: 1,
    justifyContent: 'center',
    backgroundColor: '#fff',
    alignItems: 'center',
  },
  mapStyle: { 
    marginTop: 20, 
    width: Dimensions.get('window').width,
    height: Dimensions.get('window').height,
  }
});
```
